## 🚀 使用方式

### 1. 选择智能体

根据任务需求选择合适的智能体人设：
- **开发新功能** → Engineering 团队
- **设计界面** → Design 团队
- **推广产品** → Marketing 团队
- **规划产品** → Product 团队
- **管理项目** → Project Management 团队
- **空间应用** → Spatial Computing 团队
- **特殊任务** → Specialized 团队
- **客户支持** → Support 团队
- **质量保障** → Testing 团队

### 2. 加载人设

将选定的 `.md` 文件内容作为 system prompt 加载到你的 AI 助手配置中。

### 3. 多智能体协作

对于复杂任务，可以使用 Strategy 目录中的协作文档来组织多智能体协作。
