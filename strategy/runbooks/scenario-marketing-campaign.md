# 📢 操作手册：多渠道营销活动

> **模式**：NEXUS-Micro 到 NEXUS-Sprint | **周期**：2-4 周 | **智能体**：10-15 个

---

## 场景

你要在多个渠道上发起协调一致的营销活动。内容需要针对平台特定、品牌一致、且数据驱动。该活动需要推动可衡量的获客和参与。

## 智能体阵容

### 活动核心
| 智能体 | 角色 |
|-------|------|
| 社交媒体策略师 | 活动负责人、跨平台策略 |
| 内容创作师 | 全格式内容制作 |
| 增长黑客师 | 获客策略、漏斗优化 |
| 品牌守护者 | 全渠道品牌一致性 |
| 数据分析报告师 | 性能追踪和优化 |

### 平台专家
| 智能体 | 角色 |
|-------|------|
| Twitter 互动师 | Twitter/X 活动执行 |
| TikTok 策略师 | TikTok 内容和增长 |
| Instagram 策展师 | Instagram 视觉内容 |
| Reddit 社区建设师 | Reddit 真实参与 |
| App Store 优化师 | App Store 展示（如为移动端） |
| 微信公众号运营师 | 微信公众号内容运营 |
| 小红书专家 | 小红书种草内容 |
| 抖音策略师 | 抖音短视频营销 |
| 知乎策略师 | 知乎专业内容营销 |

### 支持
| 智能体 | 角色 |
|-------|------|
| 趋势研究师 | 市场时机和趋势对齐 |
| 实验追踪师 | A/B 测试活动变体 |
| 执行摘要生成师 | 活动报告 |
| 法律合规检查师 | 广告合规、披露要求 |

## 执行计划

### 第 1 周：策略与内容创作

```
第 1-2 天：活动策略
├── 社交媒体策略师 → 跨平台活动策略
│   ├── 活动目标和 KPI
│   ├── 目标受众定义
│   ├── 平台选择和预算分配
│   ├── 内容日历（4 周计划）
│   └── 各平台参与策略
│
├── 趋势研究师 → 市场时机分析
│   ├── 要对齐的热门话题
│   ├── 竞品活动分析
│   └── 最佳发布时机
│
├── 增长黑客师 → 获客漏斗设计
│   ├── 落地页优化计划
│   ├── 转化漏斗映射
│   ├── 病毒机制（推荐、分享）
│   └── 渠道预算分配
│
├── 品牌守护者 → 活动品牌指南
│   ├── 活动特定视觉指南
│   ├── 信息框架
│   ├── 活动的语气和声音
│   └── 宜做和忌做
│
└── 法律合规检查师 → 广告合规审查
    ├── 披露要求
    ├── 平台特定广告政策
    └── 监管限制

第 3-5 天：内容制作
├── 内容创作师 → 多格式内容创作
│   ├── 博客文章/文章
│   ├── 邮件序列
│   ├── 落地页文案
│   ├── 视频脚本
│   └── 社交媒体文案（平台适配）
│
├── Twitter 互动师 → Twitter 特定内容
│   ├── 发布线程（10-15 条推文）
│   ├── 每日互动推文
│   ├── 回复模板
│   └── 话题标签策略
│
├── TikTok 策略师 → TikTok 内容计划
│   ├── 视频概念（3-5 个视频）
│   ├── 钩子策略
│   ├── 热门音频/格式对齐
│   └── 发布时间表
│
├── Instagram 策展师 → Instagram 内容
│   ├── 信息流帖子（轮播、单图）
│   ├── Stories 内容
│   ├── Reels 概念
│   └── 视觉美学指南
│
├── 微信公众号运营师 → 微信内容
│   ├── 深度文章（1500-3000 字）
│   ├── 图文消息排版
│   ├── 菜单栏配置
│   └── 自动回复设置
│
├── 小红书专家 → 小红书种草
│   ├── 笔记标题和封面设计
│   ├── 真实使用体验分享
│   ├── 话题标签策略
│   └── KOC/KOL 合作计划
│
├── 抖音策略师 → 抖音内容
│   ├── 短视频脚本（15-60 秒）
│   ├── 热门 BGM 选择
│   ├── 挑战赛策划
│   └── 直播计划
│
└── 知乎策略师 → 知乎营销
    ├── 专业问题回答
    ├── 专栏文章
    ├── 圆桌讨论参与
    └── Live 讲座策划
```

### 第 2 周：发布与激活

```
第 1 天：预发布
├── 所有内容排队并 scheduled
├── 分析追踪验证
├── A/B 测试变体配置
├── 落地页上线并测试
└── 团队简报参与协议

第 2-3 天：发布
├── Twitter 互动师 → 发布线程 + 实时互动
├── Instagram 策展师 → 发布帖子 + stories
├── TikTok 策略师 → 发布视频
├── 小红书专家 → 种草笔记发布
├── 微信公众号运营师 → 公众号文章推送
├── 知乎策略师 → 专业回答和专栏发布
├── Reddit 社区建设师 → 真实社区帖子
├── 内容创作师 → 博客文章发布 + 邮件群发
├── 增长黑客师 → 付费活动激活
└── 数据分析报告师 → 实时仪表盘监控

第 4-5 天：优化
├── 数据分析报告师 → 前 48 小时性能报告
├── 增长黑客师 → 基于数据的渠道优化
├── 实验追踪师 → A/B 测试早期结果
├── 社交媒体策略师 → 参与策略调整
└── 内容创作师 → 基于反馈的响应内容
```

### 第 3-4 周：持续与优化

```
每日：
├── 平台智能体 → 参与和内容发布
├── 数据分析报告师 → 每日性能快照
└── 增长黑客师 → 漏斗优化

每周：
├── 社交媒体策略师 → 活动性能评审
├── 实验追踪师 → A/B 测试结果和新测试
├── 内容创作师 → 基于性能数据的新内容
└── 数据分析报告师 → 每周活动报告

活动结束：
├── 数据分析报告师 → 全面活动分析
├── 增长黑客师 → ROI 分析和渠道效果
├── 执行摘要生成师 → 活动执行摘要
└── 社交媒体策略师 → 经验教训和建议
```

## 活动指标

| 指标 | 目标 | 负责人 |
|--------|--------|-------|
| 总触达 | [基于预算的目标] | 社交媒体策略师 |
| 参与率 | 全平台平均 > 3% | 平台智能体 |
| 点击率 | CTA > 2% | 增长黑客师 |
| 转化率 | 落地页 > 5% | 增长黑客师 |
| 获客成本 | < [目标 CAC] | 增长黑客师 |
| 品牌情感 | 净正面 | 品牌守护者 |
| 发布内容数量 | [目标数量] | 内容创作师 |
| 完成 A/B 测试 | ≥ 5 | 实验追踪师 |

## 平台特定 KPI

| 平台 | 主要 KPI | 次要 KPI | 智能体 |
|----------|------------|---------------|-------|
| Twitter/X | 展示量 + 参与率 | 粉丝增长 | Twitter 互动师 |
| TikTok | 播放量 + 完成率 | 粉丝增长 | TikTok 策略师 |
| Instagram | 触达 + 收藏 | 主页访问 | Instagram 策展师 |
| 小红书 | 笔记阅读量 + 互动率 | 粉丝增长 | 小红书专家 |
| 抖音 | 视频播放量 + 完播率 | 直播间观看 | 抖音策略师 |
| 微信公众号 | 阅读量 + 在看数 | 新增关注 | 微信公众号运营师 |
| 知乎 | 赞同数 + 收藏数 | 专业认可 | 知乎策略师 |
| Reddit | 赞成票 + 评论质量 | 引荐流量 | Reddit 社区建设师 |
| Email | 打开率 + 点击率 | 退订率 | 内容创作师 |
| Blog | 自然流量 + 页面停留时间 | 反向链接 | 内容创作师 |
| 付费广告 | ROAS + CPA | 质量得分 | 增长黑客师 |

## 品牌一致性检查点

| 检查点 | 时间 | 智能体 |
|-----------|------|-------|
| 发布前内容审查 | 每篇内容 | 品牌守护者 |
| 视觉一致性审计 | 每周 | 品牌守护者 |
| 语气和声音检查 | 每周 | 品牌守护者 |
| 合规审查 | 发布前 + 每周 | 法律合规检查师 |
